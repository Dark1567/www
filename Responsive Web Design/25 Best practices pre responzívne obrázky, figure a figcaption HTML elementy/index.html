dobrý nápad pri obrazkov pri responsive designe je  vždy do css pridať niekde hore tento zapis
	img{
	max-width: 100%;
	height: auto;
}
-čiže teraz obrázok sa môže zmenšovať 	ale pri zväčšivani nikdý nepresiahne tu jeho veľkosť ktorú reálne ma.

dalšia vec pri responsive design 

širku obrazku by sme mali zadávať v percentách a nie v px

ako odstraniť medzeri medzi obrazkami 

-veľmi jednoduch v css nastavime elementu (main)font-size:0; poväčšine toto vymaže vertikálne  horizontalne  tam ostanu pretože obrazky sa zarovnávajú ku riadku textu

fix pre zarovnanie zrušenie medzier u obrazkov font-size:0; a vertical-align:middle;


zapis

	img{
	max-width: 100%;
	height: auto;
	vertical-align: middle;
} /*toto je ceľkom solidny zaklad ktory sa nachadza v každom css responsive design-e*/

main img nastavime 33.333333333%;




element fiugre sa často použiva pri obrazkoch a do figure figcaption

-do elementu figure vložime element img a pod neho do figure vložime figcaption
	figcaption pridáva obrazku popis
	-dalej v css potrebujem figure nastaviť font-size: aby sme niečo videli pretože rodičovskemu elementu sme dali font-size 0; a nejaku čiernu farbu textu jeho position: bude relative; pretože figcaption k nemu napozicujeme a jemu dame position absolute; left:1em; bottom:1em; bude dole na lavej strane v obrazku
	
keďže sme elementy Img obalili do figure

	tak musíme to width:33.333333333% použiť na figure a nie na main img aby boli v troch stlpcoch
