css opacity vs css rgba();

opacity je nepriesvitnosť tzn. že opacity:0.8; tzn. že element je na 80% nepriesvitný a na 20% priesvitny zadáva sa od 0-1 kde 1=100% pri farbách sme si tiež ukazovali background alfa kanal rgba();
je rozdiel medzi opacity a rgba prehladnosti keď menim rgba()alfa kanal tak text ostava 100%ne nepriesvitny a priesvitnosť meni len pozadie. u opacity sa meni priesvitnosť aj textu aj pozadia.

ako vycentrovať na stred element s position:absolute

kedže sme classu .post-footer nastavili position absolute tak margin:0 auto ; nefunguje lebo sme element vytrhli z toku dokumentu 1) rieši sa to tak že position:absolute; left: dame 50%; 2) a však teraz je element hodeni priliš v pravo a to ztoho dôovodu že on zoberie lavý horný roh elementu a ten nastaví na 50%; 3) ak však chceme element naozaj vycentrovať musime nastaviť stredny bod elementu post-footer 4) to spravime tak že použijeme negativný margin a posunieme tento element do lava o polovicu širky elementu čiže margin-left:-350px lebo cely .post-footer ma width 700px a polovica je 350px
